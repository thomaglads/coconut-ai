## ğŸš€ Upgrade Guide

### From 0.x to 1.0.0-beta
1. **Backup Data**: Export any active datasets
2. **Uninstall Old Version**: Remove previous Coconut AI installation
3. **Install New Version**: Run Coconut AI Setup 1.0.0-beta.exe
4. **Import Data**: Load your datasets back into the new version
5. **Initialize AI**: Click "Initialize" to load Hybrid-RAG system

### Migration Notes
- **Settings**: Preferences are preserved automatically
- **Data**: All CSV files and databases remain compatible
- **Models**: Llama model is updated automatically
- **Performance**: Expect significantly faster startup and responses
